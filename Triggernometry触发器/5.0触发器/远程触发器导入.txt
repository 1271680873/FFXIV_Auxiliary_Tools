Triggernometry - Remote triggers - 右键 - Add → Repository
第一行 Repository name 随便写   (汉化翻译：仓库名称)
第二行 Address 触发器的RAW地址  (汉化翻译：仓库地址)
https://raw.githubusercontent.com/553469159/FFXIV_Auxiliary_Tools/master/Triggernometry%E8%A7%A6%E5%8F%91%E5%99%A8/5.0%E8%A7%A6%E5%8F%91%E5%99%A8/5.0%E8%A7%A6%E5%8F%91%E5%99%A8.xml

然后一定要勾选：倒数第二项的:Allow triggers to perform file operations  (汉化翻译：允许触发器操作文件)
（否则会少东西，导致无法正常工作）
